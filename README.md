# Currency Converter

A simple and responsive Currency Converter web application that converts one currency to another using real-time exchange rates.

## Project Overview
This project allows users to convert an entered amount from one currency to another using live exchange rates fetched from a public API. It is built using HTML, CSS, and JavaScript with a clean and user-friendly interface.

## Features
- Convert between multiple international currencies
- Real-time exchange rates
- Simple and responsive UI
- Error handling for invalid selections
- Uses live currency data API

## Tech Stack
- HTML
- CSS
- JavaScript
- Frankfurter Currency API

## How It Works
1. User enters an amount
2. Selects source and target currencies
3. Application fetches real-time exchange rates
4. Displays the converted value instantly
